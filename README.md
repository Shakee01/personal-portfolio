# Personal Portfolio

A responsive personal portfolio website built with semantic HTML and CSS for the roadmap.sh project.

## Pages

- Home
- Projects
- Articles
- Contact

## Features

- Semantic HTML structure
- Responsive layout
- Flexbox and CSS Grid
- Media queries
- Styled navigation bar
- Styled contact form
- SEO meta tags on each page

## Technologies

- HTML5
- CSS3

## How to run

Open `index.html` in any web browser.
